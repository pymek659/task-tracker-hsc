Getting started

Up and running
in seconds.

01
Open the tracker
Hit "Open my tracker" and you're in. Your five default HSC subjects are already set up and ready to go — no configuration needed.
02

Use the traffic light buttons on each task to rate your confidence. Red = struggling, amber = getting there, green = confident. Watch the overall bar fill up.

Jump to the Assessments page and add upcoming tasks with due dates. The circular countdown keeps you honest.
04

Use the Pomodoro timer to lock in. Sessions are tracked daily so you can see how much work you're actually putting in each day.
